## ox agent prime

Bootstrap agent session with team context

### Synopsis

Bootstrap an AI coding agent session with team context and project configuration.

Returns an agent_id and team context information for the current project.
Agents use this to load team norms, conventions, and architectural decisions.

This command is designed for fail-fast operation - it will never block
coding agents for extended periods.

```
ox agent prime [flags]
```

### Options

```
      --agent string       Agent identifier (claude-code, cursor, droid, windsurf) (default: none)
      --agent-ver string   Agent version (e.g., 1.0.42) (default: none)
  -h, --help               help for prime
      --idempotent         Skip priming if session already primed (token optimization)
      --model string       Model identifier (claude-opus-4-5, gpt-4o) (default: none)
      --review             Security audit mode: show English summary + JSON
      --text               Output human-readable text instead of JSON
```
